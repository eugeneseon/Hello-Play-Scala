Play Scala for dummies (Pre-alpha version)

This template application shows how to organize your application with angularjs, phoenix, flume.

Trying It Out

Dev Mode

  1. go to the project root directory
  2. activator
  3. run

Prod Mode

  1. go to the project root directory
  2. activator
  3. assembly
  4. go to the target directory and find the fat jar for the application
  5. java -jar yourapplicationame.jar
